Departments_tree
Веб страница, которая выводит древовидную структуру отделов со списком сотрудников.
Информация о каждом сотруднике хранися в БД и содержит следующие данные:
    1. ФИО;
    2. Должность;
    3. Дата приема на работу;
    4.ЗП
    5.Подразделение
Дерево отделов отображается в свернутом виде.
База данных содержит 50 000 сотрудников и 25 подразделений в 5 уровнях иерархий.
Управление записями CRUD через административную часть Django.


Установка:
Склонируйте репозиторий
Создайте и войдите в вирутальное окружение
Установите зависимости:
    cd departments_project
    pip install -r requirements.txt
Можно использовать демо данные или удалить миграции и базу данных(пользователь admin/1234)
Проведите миграции
    python manage.py makemigrations
    python manage.py migrate
Создайте суперпользователя
    python manage.py createsuperuser
Запустите сервер
    python manage.py runserver

Технологии:
Python              3.8.10
Django              3.2.6
django-mptt         0.12.0
